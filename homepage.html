Karonese101

What is Karonese101?

              Karonese101 is an educational website where you can learn many different things about karonese culture with many different ways such as interactive games, organized notes, and many more to come. The goal of this website is to make people understand more about karonese culture in a fun yet effective way. This website will also be made in an easy to understand way so that people who do not know much about karonese culture will still be able to understand the contents of this website.

Future of Karonese101

              As of right now, Karonese101 only has a few features which are the home page where you can read information regarding the website, the notes page where you are provided some notes about various topics within Karonese culture, and a game about the 12 karonese family relations(Perkade-kaden 12), but in the future I plan to add a lot more to this website. Some things I plan to add are more games about karonese culture, a game about karonese language, and also new features for instance a login feature and a language feature.

Background of Karonese101

              Karonese101 is a website that I made myself with the help of a few family members after deciding to make a website as my first coding project. In the middle of my summer break, I decided to pick up a new hobby. At first, I didn't know what hobby I wanted to explore, but at the end I decided to learn how to code. After learning the basics of coding, I wanted to make my first project and decided to make a website. The problem was I didn't know what to put in the website. After some discussion with my dad, we decided to make the website about karonese culture. And thats how I ended up making Karonese101.
